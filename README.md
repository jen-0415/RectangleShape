# Rectangle Shape Activity – C# Console Application

This project is a simple C# console application that demonstrates how to apply Object-Oriented Programming (OOP) concepts by creating a Rectangle class. The program allows the user to input the rectangle’s length and width, then calculates and displays the area and perimeter.

Features:
1. User input for length and width
2. Validation: prevents zero or negative values
3. Computes: Area and Perimeter
4. Displays rectangle details using a dedicated Display() method
